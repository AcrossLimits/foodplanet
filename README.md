# Food Planet Framework
An open source quiz framework based on the game Food Planet, an application built by AcrossLimits Ltd. (Malta) for the  [Europeana Food and Drink project](http://foodanddrinkeurope.eu/). All replications of the application using the provided template must follow the rules of the above mentioned license: Attribution-NonCommercial 2.0 Generic (CC BY-NC 2.0).
